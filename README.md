# LIYNMAR Online Tutorial Services

Admin dashboard for managing teachers, students, bookings, and payments for LIYNMAR Online Tutorial Services.

## Tech Stack

### Frontend
- **React 18** - UI library
- **React Router** - Navigation
- **React Query** - Data fetching
- **Vite** - Build tool
- **Axios** - HTTP client

### Backend (Coming Soon)
- **Node.js** - Runtime
- **Express** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
